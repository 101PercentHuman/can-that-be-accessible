The main colours used would be Dark Blue and Light Blue (as you can see on my logo). While it would have been fine to stick to a blue gradient, I added Dark Purple and Pale Yellow in case if I ever needed more colours.

I checked all of the colour contrast of each colour using [WebAIM Contrast Checker](https://webaim.org/resources/contrastchecker/). While I'm aiming for WCAG AA compliance across this website, I'll take the extra step and go with AAA compliance for colour contrast.

The light colours should all have sufficient contrast against Black, and all the dark colours should have sufficient contrast against White.
